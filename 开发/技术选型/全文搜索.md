## 选型

### bleve

Go实现，⭐9.1k

亮点

- 以包的形式提供
- 支持持久化索引、纯内存索引模式
- 支持多列排序、支持分页（From、Size）
- 支持自定义分词器（自带的 cjk 对中文的支持一般，可替换为 jieba 分词器，自定义词典与停用词）
- 支持直接 struct 传入
- 支持 http 接口

不足

- 无 admin 界面
- 文档页炸了

存储

- 支持文件持久化存储
- 支持纯内存索引

### gofund

Go实现，⭐1.4k

优点

- 提供 http、grpc 接口
- 支持自定义词典
- 提供二进制文件或自行编译
- 提供 admin 界面

缺点

- 分词器仅支持简单分词

存储

- 文件分片存储
