## zip

```shell
zip -r [目标文件名].zip [原文件/目录名]
unizp [原文件名].zip
```

> r：表递归

## tar

```shell
tar -cvf [目标文件名].tar [原文件名/目录名]
tar -xvf [原文件名].tar
```

> c：create 创建
> x：extract 解包
> v：verbose 详细信息
> f：filename 文件名，f后必须加文件名

## tar.gz

```shell
gzip [原文件名].tar
gunzip [原文件名].tar.gz
```

```shell
tar -zcvf [目标文件名].tar.gz [原文件名/目录名]
tar -zxvf [原文件名].tar.gz
```

> z：表使用gzip算法进行压缩/解压
