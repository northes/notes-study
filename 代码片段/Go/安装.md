
## linux

### 下载

去[官网](https://go.dev/dl)找到最新的版本

```shell
wget -c https://dl.google.com/go/go1.14.2.linux-amd64.tar.gz -O - | sudo tar -xz -C /usr/local
```

### 环境变量

`.bashrc`

```
export GOROOT=/usr/local/go
export GOPATH=/root/go
export PATH=$PATH:$GOROOT/bin
```

重载

```shell
source ~/.bashrc
```

## 验证

```shell
go version
go env
```
