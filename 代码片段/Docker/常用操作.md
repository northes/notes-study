
## 更新、修改已启动的容器信息

仅能修改cpu、内存限制、重启策略

```bash
docker stop <name_or_id>
# 修改为总是重启
docker update --restart always <name_or_id>
```

## 重命名Container

```bash
docker rename <old_name> <new_name>
```

## 查看容器、镜像信息

包括映射的路径、启动参数、Labels、端口、网络等

```bash
docker inspect <name_or_id>
```

## 查看容器与原始镜像间发生的文件变化

```bash
docker diff <container_name_or_id>
```
