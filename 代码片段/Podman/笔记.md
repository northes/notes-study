## 容器自启动

生成自启动配置

```bash
# -f 生成文件
# -n 使用容器名替代容器 ID
# -t 超时
# -new 每次启动都会删除旧容器，启动新容器
podman generate systemd --restart-policy always -t 1 -n -f redis
```

复制配置到 systemd 配置目录下

```bash
cp container-redis.service /etc/systemd/system/
```

重载配置

```bash
systemctl daemon-reload
```

设置自启动并立即启动容器

```bash
systemctl enable container-web.service --now
```
