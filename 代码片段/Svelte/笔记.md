## JetBrains 集成

安装 `Svelte` 插件

## 创建项目方式 1

```bash
npm init vite
# 选择 svelte 选项
```

## 创建项目方式 2

```bash
npx degit sveltejs/template my-svelte-project
npm install
```

从 Github 中拉取模板，模板已不再维护，推荐使用创建项目 1

### 修改端口与地址

```js
  "scripts": {
	...
    "start": "sirv public --no-clear --port 9898 --host 0.0.0.0"
  },
```

## 部署

```bash
npm run build
```

直接部署整个 public 目录
